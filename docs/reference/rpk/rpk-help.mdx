Help provides help for any command in the application.
Simply type `rpk help [command]` for full details.

## Usage

```bash
rpk help [command] [flags]
```

## Flags

  <tbody>
    <tr>
      <td>

      </td>

        <strong>Type</strong>

    </tr>
    <tr>
      <td>-h, --help</td>
      <td>-</td>
      <td>help for help.</td>
    </tr>
    <tr>

      <td>string</td>

    <tr>
