View registry of configurable prebuilt data transforms.

## Commands

- [describe](../rpk-transform-registry-describe): Describe a registry module
- [list](../rpk-transform-registry-list): List all modules in the registry.

## Usage

```bash
rpk transform registry [flags] [command]
```
